# Dogmas da Lógica 📜

O objetivo principal seria estudar e conhecer os dogmas da Igreja Católica e os fundamentos da linguagem de Programação Prolog.

Inicialmente no projeto, para cada dogma haveria um conjunto de elementos que teriam algum tipo de relação com um dado dogma. As possíveis relaçoes seriam: suporte, objeção. Na relação de suporte poderia existir argumentos biblicos, argumentos de magistério(tradição). Na relação de objeção poderia existir argumentos contrários à doutrina, se um argumento contrario for devidamente refutado ele recebe uma classificacao de heresia.

O objetivo dessas relaçoes é estruturar um mecanismo para destrinchar um dgma qualquer trancando uma arvore de decisao para mostrar os fundamentos da Fé que o sustentam. Isso significa que nas folhas dessa arvore só pode haver evidencias que não podem ser refutadas, ou seja axiomas. Além desses elementos fundamentais, gostaria de adicionar elementos não fundamentais que auxiliaem no estudo do dogma como insigths proprios, comentarios de padres, documentos oficiais da igreja, concilios, etc.

O projeto deve ser desenvolvido de forma incremental, conforme o estudo dos dogmas vai avancando. Quero que o projeto seja simples de adicionar novos dogmas, de forma que se torne uma ferramenta útil para estudos e manutenção do meu conhecimento sobre o assunto.
